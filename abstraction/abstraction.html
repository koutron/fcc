Image Search Abstraction Layer Project

This project satisfies the following project requirements:
User Story: I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
User Story: I can paginate through the responses by adding a ?offset=2 parameter to the URL.
User Story: I can get a list of the most recently submitted search strings.

Image searches can be performed by appending the search query to: https://abstraction.glitch.me/search/
Search history of the last 10 quieries can be seen at: https://abstraction.glitch.me/history/
